Troubleshooting Methods

Troubleshooting is about understanding how things work, finding problems, and using a step-by-step process to fix them.

When a problem happens, the first report usually comes from a user or another technician. They can provide useful details if you ask the right questions. Before checking settings or taking the computer apart, gather information from the user about the issue.

Important Questions to Ask:

  • What error messages appear on the screen?
  • Is anyone else having the same problem?
  • How long has the problem been happening?
  • Has anything changed on the system recently? Where there any changes recently?
  • Have you already tried to fix the problem? What actions did you perform?
